# JCI El Alia — Landing Page Branding & Technical Constraints

## Role & Objective
You are an expert Frontend Developer, UI/UX Designer, and static site architect. Your objective is to build a highly professional, beautiful, and fully responsive landing page/website for the local JCI chapter: "JCI El Alia" (Jeune Chambre Internationale El Alia, Tunisia).

## Tech Stack Constraints (Strictly Free-Forever)
- **Framework:** Tailwind CSS (imported via CDN in the `<head>`, no build step required).
- **Icons:** FontAwesome or Heroicons via CDN.
- **Hosting:** Engineered to run flawlessly on GitHub Pages as a static site.
- **Forms:** Static integration using a free handler like Formspree or a styled embedding of a Google Form.
- **Media:** Standard web formats, using clean CSS filters for consistent tone.

## Design & Visual Inspiration
Visual blueprint is the JCI International official website (https://jci.cc). Capture its premium, global leadership aesthetic:
- **Color Palette:** Dominant JCI Blue (#0096D6) as the primary brand color, supported by crisp Whites, subtle Light Greys (#F9FAFB) for section backgrounds, and Dark Charcoal (#1F2937) for professional typography.
- **Typography:** Clean, modern, trustworthy sans-serif fonts (e.g., Inter, Montserrat, or Open Sans via Google Fonts).
- **UI Elements:** High-contrast call-to-action buttons, card-based layouts with smooth hover transformations, sophisticated borders, and plenty of breathing room (white space).

## Required Content Extraction & Research
Do not rely entirely on generic placeholder text. Actively look up and research "JCI El Alia" across the web, specifically targeting their official Facebook page and JCI Tunisia national archives. Look out for, draft text for, and structure sections around their real history, local assemblies, executive board structures, and high-impact community projects, including:
- Environmental and health initiatives (e.g., local civil society collaborations, Tunisia Green Impact Day).
- Educational and psychological programs (e.g., primary school children's development initiatives).
- Inter-chapter collaborations and twinning agreements.

If specific text details are missing during lookup, write highly contextualized, professional draft text tailored exactly to a Tunisian JCI chapter's tone rather than generic "Lorem Ipsum".

## Website Architecture (Single-Page Seamless Navigation)
1. **Sticky Navbar:** Elegant blur effect (backdrop-blur), high-res branding placement for JCI El Alia, and smooth-scroll links (Home, Who We Are, Our Impact, Join Us).
2. **Hero Section:** Impactful headline reflecting the JCI mission ("Developing Leaders for a Changing World in El Alia"), a semi-transparent dark/blue overlay over a background image placeholder, and an eye-catching "Become an Active Citizen" CTA button.
3. **Core Pillars (The JCI Mission):** A 3 or 4-column modern card grid illustrating the areas of opportunity: Individual Development, Community Action, International Cooperation, and Business/Entrepreneurship.
4. **Project Showcase (Our Impact):** A clean, dynamic card layout showcasing JCI El Alia's flagship achievements extracted from research. Each card must feature an image container, a prominent badge, a project title, a short narrative, and an impact metric.
5. **Membership / Action Section:** A highly engaging section detailing how young leaders (ages 18-40) can join, featuring a beautifully styled, user-friendly intake form interface.
6. **Footer:** A professional 3-column dark footer containing chapter details, quick links, a nod to JCI Tunisia & JCI International, and social media connectivity icons.

## Execution Deliverables
Provide complete, production-ready, clean HTML5 code structure. Ensure all Tailwind utility classes are fully optimized for responsiveness (using sm:, md:, lg: modifiers) so that the experience is breathtaking on both desktop and mobile screens. Keep the code beautifully commented so it is easy to iteratively update later.
