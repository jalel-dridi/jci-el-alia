# Images

Place high-quality images here. Recommended naming convention:

- `hero-bg.jpg` — Hero section background (landscape, min 1920x1080)
- `logo-jci-el-alia.png` — Chapter logo (transparent PNG, min 512x512)
- `project-[name].jpg` — Project photos (landscape, min 800x600)
- `team-[name].jpg` — Board member headshots (square, min 400x400)
- `about-section.jpg` — Photo for the "Who We Are" section
